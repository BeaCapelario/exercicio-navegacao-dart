Este repositório contém o desenvolvimento de um exercício focado na implementação de fluxos de navegação entre telas utilizando Dart e Flutter. O projeto está sendo construído para consolidar o conhecimento sobre as principais técnicas de transição de interfaces e gerenciamento de pilhas de rotas em aplicações mobile.

---

### O que está sendo explorado:
* Rotas Nomeadas: Organização e definição de caminhos para navegação de forma estruturada.
* Navigator API: Uso dos métodos push, pop e pushReplacement para controle da pilha de telas.
* Passagem de Parâmetros: Transferência de dados entre diferentes telas da aplicação.
* Interface Multi-telas: Construção de layouts distintos que se conectam através de ações do usuário.
* Organização de Arquivos: Separação de views e componentes para manter o código escalável.
